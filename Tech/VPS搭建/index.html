在centos环境下搭建梯子

其实搭VPS不难，难的是选称心的VPS啊！！
一般现在的我对VPS要求是:

  • Annually Fee <= $30
  • 起码能跑15m的水管 像想我这种小白当然首选物美价廉(容易被墙)的BWG啦，什么Linode,Vultr,DO基本都是5刀/月起步的，玩不起玩不起。

然后刚好早上刷出BWG $19.99 套餐补货的消息。选了这个 套餐，最普通的500G流量service，8个机房可选，面板里可以另外迁移到两个CN2的机房，不过剩余流量会只有原来的1/3。
(现在普通的DC2 QN机房已是全程CN2,并且原流量不变)

选定后开始搭建VPS & 梯子。

基础环境

重装系统

在main-controls那里stop机器，然后install new os，安装CentOS7_86x64。复制密码准备登录主机。密码和端口也会发到你的邮箱。

登录主机

下载并使用pputy远程登录vps。
查看自己的系统和内核信息用 

uname -a

先检查系统更新 

yum update -y

魔改bbr加速

搬瓦工KVM自带的有后缀bbr的系统，是直接启用了bbr加速，但是没有经过魔改的bbr速度不及锐速，所以个人还是推荐锐速。同时bbr只支持4.9以上4.13以下的内核，锐速不支持4.9以上的内核，所以不能共存。但他们都可以和kcptun共存的。

魔改bbr一键脚本

第一步安装最合适的内核，这里都指定v4.11.9

yum install -y wget && wget --no-check-certificate -O C71.sh https://raw.githubusercontent.com/xratzh/CBBR/master/C71.sh && sudo bash C71.sh

安装新内核后，输入Y，系统会自动断开putty，重启一次，重新登录后启用新的内核。

第二步使用以下的脚本编译开启魔改的bbr。

wget --no-check-certificate -O C72.sh https://raw.githubusercontent.com/xratzh/CBBR/master/C72.sh && sudo bash C72.sh

结束后，显示Finish表示正常，或者执行

lsmod |grep 'bbr_powered'

结果不为空,则加载模块成功。

装SSR

只安装SSR，直接一键 

wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh
chmod +x shadowsocksR.sh
./shadowsocksR.sh 2>&1 | tee shadowsocksR.log

会让你输入密码和端口(0-65535之间的数)，选择加密和混淆方式。
卸载SSR

./shadowsocksR.sh uninstall

查看SSR状态

/etc/init.d/shadowsocks status

也可以使用teddysun的各种版本的SSR的一键脚本，详情：https://shadowsocks.be

切换机房

当察觉ip被墙或网络不好，抽风时候，可以在面板切换机房。

一键添加500swap:

    dd if=/dev/zero of=/swapfile bs=1M count=512 && \
    chmod 600 /swapfile  && \
    mkswap /swapfile  && \
    swapon /swapfile  && \
    cp /etc/fstab /etc/fstab.bak && \
    echo '/swapfile none swap sw 0 0' | tee -a /etc/fstab && \
    sysctl vm.swappiness=10 && \
    sysctl vm.vfs_cache_pressure=50 && \
    echo vm.swappiness = 10 >> /etc/sysctl.conf && \
    echo vm.vfs_cache_pressure=50 >> /etc/sysctl.conf && \
    sysctl -p
