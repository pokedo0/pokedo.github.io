工作经历

 2019-09-13 2020-12-31 11 minutes read (About 1639 words)

记录下自己毕业后的工作经历，对之后再找工作或者是有需要面经的同学都有帮助。

由于可能涉及个人隐私问题，会隐藏部分信息

1. 广州壬丰大厦处

  • 2019.08 - 2019.09

第一份工作，毕业后找了一个月后入职，面试要求不高，会Spring系列+ spring cloud 微服务 + 前端框架即可。

公司环境还行，包了整层，上班时间 8.5 - 5.5，周六日隔周错开调休，中午包餐，快速解决。因为公司都是年轻人为主，下班基本到点准时走人，气氛还是比较自由的哈哈。

薪资试用期给的4k，转正6k。问身边一同事，刚进来3.5k，工作快两年，没10k薪资，说是准备年末走人。

工作是对口的Java Web 开发，团队内需要解决旧架构积累的Bug问题，然后逐渐过渡到新架构 Spring Cloud + Vue。

开发前期我们需要用Eclipse去运行旧框架 GWT 项目，真的贼恶心，里面是08年开始维护的始祖代码，全程就是写类似于Swing的GUI界面，Web页面上的button什么的需要全部写在Java代码中，还要监听事件写逻辑；数据库也是用最原始的JDBC+工具类解决。真的一个成熟的Web框架都没有。

凑巧的是，我刚进来2周左右，竟然遇上同班同学，，感觉有人陪伴 摸鱼 真好，同学试用给的4.8k，转正应该有6k多。

后来等下家的Offer确定后，就在8月末辞职了，休息一周后入职。祝愿我同学能早日接手新架构项目的开发。

2. 广东汇丰软件开发 - 太古汇一座

  • 2019.09 - Now

第二份工作（太快了吧~ ，面试在上家集中面试期间一起面的，因为是大公司不想错失机会，因此入职上家后还头铁地请假去面试。

面试8成靠英语，可能我面的职位职责有偏差，不是单纯的Java开发，也包含维护的职责，问的技术问题并不深入，没有涉及到任何开发框架问题，反倒是Sql语句，Linux语句，Web开发流程等基本技术问题多点，自己遇到不懂的问题，我也尝试用英语表达自己的理解。最后面试官问我能不能接受其他相关工作时内心就有点落差，还问我接不接受早班（记得看清楚申请的Title职责，看是否能否接受，我的是Associate Software Engineer, band 7

总之汇丰的面试，能说英语尽量英语，次之粤语（我是全程英语的，这也意味着你必须准备好自己的英文稿子，去看准网搜集问题准备自己一套英文答案，比如 what expection from the job之类的工作规划问题都会有问道。

之后走的三方背调流程，请的是First Advantage公司，填表格时对于工作经历，如果是在职状态，里面有提供填写可以联系公司的日期，在此之前辞职就可以了。大概在填好表格的2-3个星期后就会开始正式背调工作，而调查到公司时会发邮件询问你，这样就可以知道背调进度了。

联系邮件

如果有想隐瞒的工作经历，先查社保，公积金有没有相应的公司缴纳记录再决定。

入职当天，先去二座排队办入职手续，上楼拍照办员工卡，领个人物品，最后去之前联系好的Manager告知的办公地点。

但是，当天我跟另一位 GBM 部门的同事一直没人认领,最后才有几位印度佬过来领人。后面才知道在确定offer之后如果没Manager联系你的话，请主动跟招聘的HR联系要上头Manager的联系方式，方便入职。

薪资方面给的8k，试用期两个月。（起码比上家double了

一流办公环境，弹性上班时间，9 - 6 双休，最新配置电脑 16G + i7 触屏的 Thinkpad，双显示屏或长曲屏，有自己饭堂，15r以上一餐，有每周固定的High tea时间，就是team内轮流几个人出钱买小零食开吃。

岗位方面上文提过，毕竟是外企，薪资也够，有点偏差还能接受吧哈哈，镀个金。个人感觉该岗位负责公司系统的维护工作，进来一周后主要是提之后各项工作需要的权限，因为里面文档没有更新，就这些准备工作都搞了一周，无语。

一些不满意的方面：

  • 弹性上班时间，不少人是9.30 - 10点来上班的，然后他们也会比较晚下班，所以到6点时间我也看身边好多人没走，不知不觉自己就拖个10-20分钟再下班，毕竟自己是新人。

  • 节假日没调休，周六日正常休息，可能拼不到长假一起放。

  • 工位略挤，一座办公还好，长桌放三个工位，隔壁二座基本是一个单位的桌子放两个工位。个人理想的办公位置还是倾向于有独立挡板的带小柜子的传统办公桌，这样更有安全感，也容易集中干自己的事情。

  • 工作流程繁琐严格，上文提到，光是提工作权限就花了一周时间，公司的电脑有配套自己一套安全软件，黑名单上网，软件有指定的商店下载安装（软件安装也要raise请求。

Anyway, 可以进到大外企已经很不错了，虽然岗位跟我想象中有点偏差，还是希望能好好干下去，学到更多实用的知识。: )
