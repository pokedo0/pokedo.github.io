19年末爆款轻薄本？ —— 联想小新Pro13

 2019-10-05 2020-12-31 13 minutes read (About 2007 words)

为什么选轻薄本？

简单的说法，以前的笔记本厂商挤牙膏太厉害了，不买游戏本的话想弥补性能上的差距太难了，于是消费者只能在主流价位上做出选择，在便携性和审美外观方面等都作出了一定的牺牲。

但现在的笔记本市场，在牺牲少些性能和散热能力基础上，消费者有更多权衡和选择笔记本的便携性和外观审美的权力。

以前是你只能二选一，但现在以同样的价钱，在牺牲些许性能 / 便携 / 外观 情况下，笔记本的日常使用体验会让消费者感到更满意。

为什么要选小新Pro？

先谈一下陪伴了我四年多的游戏本 - 华硕飞行堡垒，5k+的价位，i5四代U + 950M，但完全被隔后一年戴尔出的暗影精灵吊打 i5六代U + 128 + 1T + 960M，也是5K+的价位，只能说，在我印象中六代U出来以前，各笔记本厂商真的是一点长进都没有，全在硬件上面挤牙膏，让消费者买单。

不过现在电商销售渠道在日常生活中已经非常普及了，稍微理智一点的消费者都会上知乎，贴吧扫一波盲，在挑本时让那些垃圾本无所遁形。

当然，华为，小米等互联网厂商的进军，俨然让笔记本领域变成一片红海，连现在的硬件屠夫神船也要重新衡量一下自己定位了，如今的消费者在笔记本硬件尚可的条件下，会重新考虑笔记本除硬件配置以外的其他因素。

所以，当提起19年的爆款机型，不再是传统的笔记本厂商的产品，而是华为的在经历前几代产品后推出的革命性MateBook系列，超高的屏占比，2K屏幕，3:2比例，凭借极高的屏幕素质迅速抓住了消费者们的眼球，毕竟第一印象很重要嘛~ 硬件配置方面采用 8代U + MX150 / 250的主流轻薄本配置，Type-C充电，价格6K+，嗯，真香 :)

老实说，要不是听到十代U出来的消息，我立马就入手MateBook 14了，可惜华为因为今年贸易战的关系，MateBook系列迟迟未有更新的消息，所以便转向了小新Pro的怀抱。

在互联网厂商的掺和下，后来有一家传统厂商 美帝良心 终于开窍了，奋起直追，如今的游戏本领域，轻薄本，万金油本领域的前排都榜上有名，游戏本排行更是长期霸住一哥的位置。。( 身边不少人同样入的这款

不要随便联想

终于等呀等，等来了大佬的一篇文章 https://zhuanlan.zhihu.com/p/80470825 ，虽然不是华为的3:2屏幕，但是看文章说小新pro是16:10比例2K屏幕，占屏比也挺高，没有黑下巴 (在此之前16:9的黑下巴轻薄本太丑了:( )，硬件方面 10代U + MX250 + Type-C充电，看大佬的说辞也很看好这台笔记本，于是加入了粉丝群蹲首批货。

外观部分

2.5K分辨率屏幕，16:10比例，极高屏占比，没啥好说的，爽就完事了。
另外16:10对于办公，游戏真的两不误，我玩的OW和L4D里面都有提供16:10比例的选择，真香 :)

硬件部分

由于我不是专业的，就直接列纸面上的参数吧，i5 十代U (14nm)，MX250显卡，16G内存，512G固态，少了点。

关于性能释放方面，我参考的评测说因为硬件上散热能力较强，性能释放方面自然有充足的发挥空间。

单拷FPU功耗稳定在35w频率3.36GHz，温度为89度。
整体稳定在767分附近可见R15跑分性能释放较强。

游戏部分

个人不玩3A大作，基本只玩些老的单机游戏和端游。

以守望先锋 OW 为例，2K分辨率下，画质选中，基本可以维持60FPS，属于流畅能玩的水平。

注意渲染比例只能选自动比例，大约为50%-60%的比例，75%比例即使在低画质下也会有卡顿情况。

接口 配件部分

小新Pro 配置了2个全能Type-C接口 ( 都支持DP协议，视频输出等 ) + 1个USB-A接口，基本上都是拓展坞或者自行换带Type-C接口的外置设备 两种方案二选一。

我个人是买了绿联的拓展坞 + 换带Type-C接口U盘。

好消息是，市面上专供MBP款的拓展坞，小新Pro都能用，这样看起来笔记本更有一体感。

拓展坞 + 充电头

一些遗憾

任何一款产品都不可能十全十美的，尤其是联想这款上来就16:10屏幕，首发国行良心价格，打头阵冲爆款的产品。（等等党大胜利

我列出以下的一些不足：

  • 玩游戏时表面温度高，或许是键程短的缘故，WASD键的位置真有点烫手，建议大型游戏选手入个散热支架。
  • 音响位置在机器底部，反正我觉得有点怪怪的，外放声音的时如果手放键盘，会明显觉得声音有变化。
  • 首发机器只有512G固态，不知道后面有没有1T版本。
  • 只有13寸的机器可选，大屏幕选手不习惯。
  • USB-A接口只有一个，鼠标选手接了这个口就没了。
  • 电源适配器是那种小方块，不便携带，我自己买了个紫米65W的两脚充电头，120+大洋不便宜。

总结

联想小新Pro是一款联想明显想冲爆款的产品，虽然整体配置和价格都非常不错，但不知道是备货还是其他原因，发售方面还是有一些可圈可点的地方：

  • 首发配置中i7跟i5版本能相差近900块的价格
  • 都上5-6k的价位了，隔壁的Air系列都有1T固态了，我觉得再加钱来个1T的版本也完全可以呀
  • 首发渠道真的被骂成狗了，特别是狗东，不少人说到7号才发货。。我还好，蹲群里抢到官网提前首发。

借用评测里的结语来结束吧：

结语：

这台笔记本电脑充分的照顾到了低价位的消费者的“全能”需求，在做到CPU性能释放极强的情况下，支持了PD充电协议，搭载了高素质的屏幕，以及合理的配置搭配。

缺货应该是这台笔记本电脑需要解决的一个问题。
